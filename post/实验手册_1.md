# 平台文档

本平台后端基于Django

未来将会在Github上开源

定义/为项目根目录

文档通过Markdown格式进行书写，放置于/post目录下

服务器读取/post目录下的文档，将内容发送至客户端并于前端进行渲染

## 文档命名规则

模块首页命名:

    模块名.md

其余页面命名:

    模块名_编号.md

例如：实验手册_0.md

模块页面内部的链接命名:

    /doc/文件名

例如：/doc/实验手册

注:文件名应当不带有.格式后缀


## 文档题目接口

发送的POST信息如下所示
{
    "Type":0/1,
    "num":0,
}

规定0页为目录
